Master Resume
Guide.

The AI needs a structured JSON "source of truth". Follow this format for best results.

Key Tips


1. Quantified Metrics

Every project/experience bullet should have a number (%, $, count, time).

2. Project Narratives

Include a "master_narrative" and "core_actions" list. The AI will weave these into unique bullets.

3. JSON Structure

Paste the sample on the right into your local file and fill your details.



Sample JSON Format

{
  "personal_info": {
    "name": "Your Full Name",
    "location": "City, State",
    "email": "email@address.com",
    "phone": "+91 0000000000",
    "linkedin": "linkedin.com/in/username",
    "github": "github.com/username"
  },
  "professional_summaries": [
    "Experienced Data Analyst specializing in Python and SQL...",
    "Aspiring ML Engineer with a focus on NLP and Deep Learning..."
  ],
  "skills": {
    "languages": ["Python", "Java", "SQL"],
    "frameworks": ["FastAPI", "React", "TensorFlow"],
    "data_science": ["Pandas", "Scikit-learn"]
  },
  "experience": [
    {
      "company": "Tech Corp",
      "role": "Data Analyst Intern",
      "start_date": "Jan 2024",
      "end_date": "May 2024",

        "Reduced query latency by 45% using optimized SQL indexing.",
        "Delivered 12+ BI dashboards to executive leadership."
      ]
    }
  ],
  "projects": [
    {
      "name": "Project Name",
      "github_link": "github.com/username/project",
      "tech_stack": ["Python", "FastAPI"],
      "master_narrative": "Detailed story of your project...",

        "Saved 20 hours a week.",

# 🚀 TailoredResume.ai (Web Suite)

**TailoredResume.ai** is an advanced, AI-driven RESUME tailoring system designed to optimize professional profiles for modern Application Tracking Systems (ATS). This web solution provides an interactive interface to transform standard master resumes into high-impact, job-specific documents that prioritize quantifiable metrics and keyword alignment.

## ✨ Key Features

- **Interactive Web Interface**: Upload your master resume JSON and paste a Job Description to get an optimized resume in seconds.
- **ATS Score Optimization**: Achieves 95%+ target scoring through precise metric injection and keyword alignment.
- **Iterative Refinement**: Automated scoring and re-generation loop to reach peak performance.
- **Quantification Engine**: Automatically converts vague descriptions into measurable results.
- **Linguistic Variance**: Eliminates repetitive action verbs to provide a professionally diverse tone.
- **Instant PDF Export**: High-fidelity, ATS-readable PDF generation via ReportLab.

## 🛠️ Tech Stack

- **Frontend**: Vanilla Modern HTML5/CSS3 (Glassmorphism & Micro-animations)
- **Backend**: FastAPI (Python 3.12+)
- **AI**: Google Gemini (Direct logic integration)
- **PDF**: ReportLab (High-fidelity minimalist templates)

## 🚀 Quick Start

### 1. Prerequisites
- Python 3.12 or higher
- A Google Gemini API Key

### 2. Installation
```powershell
# Clone the repository
git clone https://github.com/krish1440/TailoredResume.ai.git
cd TailoredResume.ai

# Install dependencies
pip install -r requirements.txt
```

### 3. Configuration
Create a `.env` file in the root directory and add your API key:
```env
GEMINI_API_KEY=your_api_key_here
```

### 4. Run the Application
```powershell
python main.py
```
Open `http://localhost:8000` in your browser.
